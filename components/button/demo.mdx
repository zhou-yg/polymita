# Button 按钮

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑

<RButton type="primary" onClick={() => console.log('click on primary')} >Primary Button</RButton>

<RButton type="text" >Text Button</RButton>

<RButton >Default Button</RButton>

<RButton type="link" >Link Button</RButton>
