# Button 按钮

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑


<Component type="primary" onClick={() => console.log('click on primary')} >Primary Button</Component>

<Component type="text" >Text Button</Component>

<Component type="link" >Link Button</Component>

<Component >Default Button</Component>



disabled 按钮展示


<Component disabled type="primary" onClick={() => console.log('click on primary')} >Primary Button</Component>

<Component disabled type="text" >Text Button</Component>

<Component disabled type="link" >Link Button</Component>

<Component disabled >Default Button</Component>



## 代码示例

```javascript
import ButtonModule from '@polymita/ui'/button'

const Button = renderToReact(ButtonModule)

export default () => {
  return (
    <div>
      <Button>Primary Button</Button>
    </div>
  )
}
```
