# Menu 菜单

为页面和功能提供导航的菜单列表

<Component items={signal2([
  {
    label: 'Menu Item One',
    key: 'key1',
  },
  {
    label: 'Menu Item Two',
    key: 'key2',
  },
])} />



## 嵌套子菜单

<Component items={[
  {
    label: 'Menu Item One',
    key: 'key1.1',
    children: [
      {
        label: 'Child Item',
        key: 'child1',
      }      
    ]
  },
  {
    label: 'Menu Item Two',
    key: 'key2.2',
  },
]} />

最多支持2层的嵌套子菜单
